QtRecon provides keyboard shortcuts to speed up common operations during reconnaissance and testing workflows.

## File operations

<ResponseField name="Ctrl+N" type="keyboard">
  Create a new workspace/project
</ResponseField>

<ResponseField name="Ctrl+O" type="keyboard">
  Open an existing workspace file
</ResponseField>

<ResponseField name="Ctrl+S" type="keyboard">
  Save the current workspace
</ResponseField>

<ResponseField name="Ctrl+Alt+S" type="keyboard">
  Save-as the current workspace to a new file
</ResponseField>

<ResponseField name="Ctrl+Q" type="keyboard">
  Exit QtRecon
</ResponseField>

## Configuration and settings

<ResponseField name="F10" type="keyboard">
  Open settings/configuration dialog
</ResponseField>

<ResponseField name="F5" type="keyboard">
  Reload configuration from file
</ResponseField>

## Scanning and importing

<ResponseField name="Ctrl+I" type="keyboard">
  Import Nmap XML file(s)
</ResponseField>

<ResponseField name="Ctrl+H" type="keyboard">
  Scan host or IP range
</ResponseField>

## Search and navigation

<ResponseField name="Ctrl+F" type="keyboard">
  Search string in notes and tabs
</ResponseField>

## Tips for efficient use

- Use **F10** frequently to adjust autorun settings and tool configurations as your methodology evolves
- **Ctrl+I** supports importing multiple XML files at once for batch processing
- The host list filter supports real-time search - just start typing IP addresses or hostnames
- **Ctrl+F** searches across both notes and application tab outputs for comprehensive coverage
