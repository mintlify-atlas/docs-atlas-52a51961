    Install QtRecon and launch your first automated reconnaissance scan
  </p>
  
  <Steps>
    <Step title="Install dependencies">
      QtRecon requires Python 3.7+ and PySide6. Install using pipenv:
      
      ```bash
      pipenv install
      ```
    </Step>
    
    <Step title="Launch QtRecon">
      Start the application:
      
      ```bash
      pipenv run python qtrecon.py
      ```
    </Step>
    
    <Step title="Configure your tools">
      Set up nmap, terminal emulator, and your favorite security tools in the configuration file at `~/.config/qtrecon/conf.json`
      
      <Tip>QtRecon comes with sensible defaults. You can start scanning immediately and customize later.</Tip>
    </Step>
    
    <Step title="Run your first scan">
      Create a new scan, enter your target IP or range, and watch QtRecon automatically execute reconnaissance tools based on discovered ports.
    </Step>
  </Steps>
